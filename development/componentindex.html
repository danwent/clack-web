Class Name Description Details
Interface Devices      
  FromDevice Reads incoming packets from a network interface [details]
  ToDevice Sends outgoing packets to a network interface [details]
Header Strip/Encap      
  EtherStrip Removes the Ethernet header from a frame  
  EtherEncap Encapsulates a payload in an Ethernet frame  
Layer Demultiplexers      
  Level2Demux Demultiplexes ARP and IP Packets [details]
  Level3Demux Strips the IP header and demultiplexes TCP, UDP, and ICMP packets [details]
ARP Layer      
  ARPDemux Demultiplexes ARP Requests and Replies. [details]
  ARPRespond Responds to incoming ARP requests [details]
  ARPLookup Contains ARP cache and handles querying for unknown hardware addresses [details]
IP Processing      
  IPHeader Check Verifies the contents of an IP Header. [details]
  IPRouteLookup Performs a longest-prefix-match lookup in a simple IP routing table [details]
  IPTTLDec Attempts to decrement the Time-To-Live field of the IP header. [details]
ICMP Processing      
  ICMPDemux Demultiplexes local ICMP packets based on Type field [details]
  ICMPEcho Responds to ICMP echo requests [details]
  ICMPTTLExpired Generates ICMP TTL-expired packets [details]
  ICMPPortUnreach Generates ICMP Port-unreachable packets [details]
Queues      
  ByteQueue A simple drop-tail FIFO queue measuring its size in bytes [details]
  REDQueue A queue implementing the Random Early Detection (RED) queuing mechanism [details]

 

Auxiliary      
  Delay Slows down packets passed through it by a configurable percentage of router speed [details]
  Loss Drops packets at a configurable rate, or at user request [details]
  Tee Duplicates packets on a single input and sends them out two outputs [details]
  Capture Dumps Packets in PCAP file format [details]
