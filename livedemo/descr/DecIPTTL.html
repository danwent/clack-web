DecIPTTL

(Documentation adapted from Click website: http://www.pdos.lcs.mit.edu/click/doc/DecIPTTL.n.html)

Clack element; Attempts to decrement the Time-To-Live field of the IP header.

PROCESSING TYPE

Push

DESCRIPTION

Expects IP packet as input. If the ttl is <= 1 (i.e. has expired), DecIPTTL sends the packet to one output port (ordinarily connected to an ICMP error packet generating element). Otherwise it decrements the ttl, re-calculates the checksum, and sends the packet to the other output.
