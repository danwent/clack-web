ICMPPortUnreach

Clack element; Generates ICMP Port-unreachable packets

PROCESSING TYPE

Push

DESCRIPTION

Accepts incoming TCP / UDP packets used to generate an ICMP destination port unreachable (ICMP Type = 3 Code = 3) messages to the source address contained in the packet. The error packet will include (as payload) the original packet’s IP header and an initial segment of its IP payload. The ICMP packet is encapsulated in an IP packet and sent to the output port.
