Delay

Clack element; Delays packets by a configurable integer factor

PROCESSING TYPE

Pull

DESCRIPTION

Has one input and one output port, both of which are pull ports. This element never initiates a pull, but rather only pulls on its input port when a connected elemented has pulled on its output port.

A Delay element delays by packets by an integer factor N, by providing valid packets only every Nth time it receives a pull request. The value of N can be dynamically configured.
