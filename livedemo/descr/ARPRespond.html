ARPRespond

(Documentation adapted from Click website: http://www.pdos.lcs.mit.edu/click/doc/ARPResponder.n.html)

Clack element; Responds to incoming ARP requests

PROCESSING TYPE

Push

DESCRIPTION

Responds to incoming ARP Requests (stripped of Ethernet header) for any IP address associated with this virtual host. Generates an immeadiate ARP Reply packet encapsulated in an Ethernet header.
