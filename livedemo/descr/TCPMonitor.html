TCPMonitor

Clack element; Monitors TCP flows encapsulated within IP packets.

PROCESSING TYPE

Push

DESCRIPTION

Has one input and one output port, both of which are push ports. This element expects IP packets on its input and passes these packets to its output unchanged.

For each IP packet, the TCPMonitor checks if encapsulates a TCP Packet and if so stores a record for a flow defined by the 5-tuple: (tcp, host A IP address, host A port, host B IP address, host B port). The assigned values of host A and B are completely arbitrary.

TCPMonitor reports key statistics about all of the flows seen recently (flows timeout after a specified period). Information includes the control values seen within the last packet sent by A and B, and the outstanding (ie: unacknowledged) data each host has sent in the network.
